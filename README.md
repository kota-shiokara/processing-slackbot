# processing-slackbot
A slack bot that turns a processing sketch into a GIF or png

# はじめに
このbotはslack上で稼働し、「processingのスケッチを代行して画像化もしくはGIF化しチャンネルに出力する」ということを目的にPython3で書かれたものです。
processing-java、pyhon3が実行できる環境である必要があります。
